This whole preview page is the demo. Its own Layout Side sidebar is responsive: a preview script sets the real OSUI .desktop/.tablet/.phone class on <html> by viewport width (as the runtime does), and the shipped tokens/outsystems-ui-side-responsive.css does the rest — no parallel system.
