## Conventions observed

- Widgets carry `data-*` hooks (e.g. `data-button`, `data-input`, `data-dropdown`).
- Common layout/utility classes: `OSFillParent`, `OSInline`, `ThemeGrid_MarginGutter`, `margin-left-s`.
- Runtime IDs are auto-generated and non-deterministic.
